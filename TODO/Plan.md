- сохранять задачи в файл;

## Пользование программой 
#### Сценарий использования:

- пользователь запускает программу через консоль;
- программа загружает задачи из файла;
- отображается список задач и меню действий;
- пользователь может:
  - добавить новую задачу;
  - отметить задачу выполненной;
  - посмотреть все задачи;
  - посмотреть выполненные задачи;
- при выходе из программы задачи сохраняются в файл.
